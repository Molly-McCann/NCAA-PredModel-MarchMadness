This is the git workflow
Committing and Pushing
1 - add any changes to be tracked by git by running
    ~ git add . 
    ~ git add myFile.py
2 - commit those change to you Local branch
    ~ git commit -m "commit message"
    ~ git commit 
        - opens vim
        - press i to insert you message
        - once done, press esc to get out of insert
        - then type :wq to write and quit
3 - push your local branch to the upstream origin
    - git push

Merging
You are currently on your local feature branch
1 - make sure your working tree is clean (you are committed and pushed)
2 - switch to main and make sure that you are up to date with all of the changes    
    ~ git checkout main
    ~ git pull
3 - switch back to feature branch and merge in any changes from main    
    ~ git checkout feature-branch
    ~ git merge main
4 - switch back to main and merge in your changes
    ~ git checkout main
    ~ git merge feature-branch
